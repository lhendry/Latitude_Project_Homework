Summary: Lattitude vs. X

The purpose of this project was to analyze the changes to the weatger as you get closer to the equator. This analysis required that we pull data from openWeather Map API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. Factors reviewed include: temperature, cloudiness, wind speed, and humidity. This site provides the source data and corresponding visulizations created as part of the homework.
